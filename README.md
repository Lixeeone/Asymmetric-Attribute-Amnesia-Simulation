# Feature Binding without Attentional Glue

This repository contains the simulation code and representational matrices for the paper: 
**"Feature Binding without Attentional Glue: An In Silico Lesion Study on Asymmetric Attribute Amnesia in Bimodal Systems"**

## Authors
Zilin Li (Shanghai Jiao Tong University) & Tianle Jiang (Sichuan University of Arts and Science)

## Overview
This project investigates the geometric degradation of representational manifolds under simulated attentional load. We utilize a digital lesion operator $\alpha \in [0, 1]$ to ablate cross-modal binding energy and measure the resulting manifold collapse ($D_{eff}$) and RSA trajectories.

## Requirements
- Python 3.8+
- PyTorch
- CLIP / DINO (pre-trained weights)
- Scipy & Matplotlib
